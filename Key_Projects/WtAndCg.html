Project Overview:

July 2023 - Oct 2024

➣ A project as an FEA Engineer in association with Blue Hat Solutions Private Limited, Bangalore, India.
➣ Automated weight and C.G Calculations for Mechanical Products.
➣ This streamlined approach, considering various combinations of store, has significantly contributed to iterative design processes.

Role and Responsibilities:

➣ Developed Python code to automate weight and C.G Calculations for Mechanical Products, introducing a streamlined approach that considered various combinations of stores.
➣ Significantly contributed to iterative design processes by providing accurate and efficient calculations through Python automation, enhancing overall design efficiency.
➣ Collaborated with cross-functional teams to identify, prioritize, and implement automation solutions, contributing to a more efficient and agile work environment.

Results and Outcomes:

➣ Developed and implemented Python code to automate weight and mass balance calculations for Mechanical Products, introducing a streamlined approach for handling various combinations of stores.
➣ Workflow Optimization: Implemented robust automation processes to streamline overall data processing workflows, reducing manual efforts and improving efficiency.

Key Skills:

➣ Python(pandas, numpy, matplot), Data Extraction, Data Manipulation, Data Processing, Data Visualization, Weight and C.G Calculations
