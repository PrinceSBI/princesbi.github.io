Bangalore House Price Prediction and Deployment

Project Overview:

Feb 2022 - April 2022

Created a Machine Learning model to predict the price of houses in Bangalore. The model was deployed on a website using Flask.

Role and Responsibilities:

➣ Performed EDA on data, such as Data Cleaning, imputing missing data, outlier removal, feature reduction, etc.
➣ Trained and Tested Linear Regression, Decision Trees, Random Forest Regressor Models. Preformed Cross Validation and Hyper parameter Tuning using GridSearchCV to obtain the best ML model
➣ Used Python Flask server as the backend to deploy the ML Model. HTML, CSS, JavaScript was used to make UI

Results and Outcomes:

➣ Successfully trained Linear Regression, Decision Trees, Random Forest Regressor Models. Linear Regression Model was found to be the best model with an score of 84.78%
➣ Developed a web application using Flask and deployed the ML model in it.
