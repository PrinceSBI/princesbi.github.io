Aerodynamic Load Transfer from CFD to FEM Grid

Project Overview:

Nov 2023 - Dec 2023

➣ A project as an FEA Engineer in association with Blue Hat Solutions Private Limited, Bangalore, India.
➣ Led a comprehensive project focused on developing a Python code to efficiently extract and transfer aerodynamic loads from CFD to FEM grid, streamlining the data processing pipeline.
➣ Integrating fluid dynamics simulations (CFD) with Structural analysis (FEA) to simulate and analyze fluid-structure interaction phenomena.

Role and Responsibilities:

➣ Conducted data extraction from CFD simulations stored in .vtk files and FEM model in .bdf files.
➣ Aligned coordinate systems between CFD and FEM grids to ensure accurate integration.
➣ Developed a Python script to extract, process, and manipulate data in array format.
➣ Implemented coordinate transformations, ensuring consistency in spatial representation.
➣ Integrated extracted data into the simulation workflow for fluid-structure interaction analysis.

Results and Outcomes:

➣ Successfully transferred forces from CFD and FEA grids, preserving total forces and moments.
➣ Achieved a streamlined workflow for fluid-structure interaction simulations, contributing to improved simulation accuracy.

Key Skills:

➣ Python(sklearn, pandas, numpy, matplot), FEM, CFD, Data Science, Data Extraction, Data Manipulation, Data Processing, Data Visualization
